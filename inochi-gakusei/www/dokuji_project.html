inochi学生プロジェクトは自らも課題解決を行ってまいりました。多くの若き学生たちと共にinochi学生フォーラムを開催するだけでなく、自分たちのヘルスケアの課題意識を主体的にアウトプットしてゆくプロジェクト「inochi独自プロジェクト」を展開しています。

2017 Project

inochi Girls’ Project
〜ほぐれて八木さん〜

2017年度の独自プロジェクト”inochi Girls’ Project” 「今社会で広まっているストレス対策教材は、本当に意味あるのだろうか。」という課題意識から、ストレス対策にもターゲティングが必要だと考え、働く女性を対象にしたストレスマネジメントe-learning教材「ほぐれて、八木さん」を作成、複数の企業に提供、教材受講効果測定を行ってきました。 inochi学生・未来フォーラム2017にて約600名の聴衆の前でプロジェクトのプレゼンテーションと啓発パネルディスカッションを展開。 「個々人に合わせたストレス対策」の普及によって、本当に効果のあるストレス対策を社会に実装していくことを目指しました。

2015 Project

AED使用/心肺蘇生×VR×若者の
医療啓発プロジェクト。

2015年度の独自プロジェクト”AED360PROJECT” AEDに関心のない層を対象にしたVR動画コンテンツを作成。 360°カメラで撮影したAED使用シーンの再現動画を登場人物視点で楽しむことができる。 2015年10月31日 inochiフェスタ・11月1日 inochiフォーラムにて、㈱ハコスコの協賛のもとブースに出展し、約150名が体験。 この動画は「ギズモードジャパン」など、ウェブメディアにも取り上げられた。
